# Gussing-Game-
 Advanced Password Guessing Game (Tkinter)
An interactive word-guessing game built with Python and Tkinter, featuring multiple categories, difficulty levels, hints, lives, a timer, a leaderboard, and theme toggle support. Designed for fun and to improve logical thinking under time pressure.

✨ Features
🎯 Multiple Categories — Choose from Animals, Countries, Food, Tech.

⚡ Difficulty Levels — Easy, Medium, Hard with different lives & time limits.

💡 Hint System — Reveal random letters (score penalty applies, limited per round).

❤️ Lives System — Limited attempts per round based on difficulty.

⏳ Timer & Progress Bar — Beat the clock for bonus points.

🏆 Leaderboard — Saves and displays top scores from all players.

🔄 Multiple Rounds — Play up to 5 rounds in a single session.

🌗 Theme Toggle — Switch between Dark Mode and Light Mode instantly.

📜 Guess History — Track all previous guesses with positional hints.
